**Announcing DCP: Saudi-focused Containerized GPU Compute Marketplace**

We're building the infrastructure for AI compute in Saudi Arabia — and today DCP is now open to providers and renters.

**What is DCP?**

DCP connects GPU owners with AI developers and researchers. If you have an NVIDIA GPU, you can list it on DCP and earn based on completed, settled jobs. If you're building AI applications and need GPU access, DCP provides LLM inference, image generation, and model training through REST APIs, billed and settled in SAR in a Saudi-focused routing model.

**Why this matters for Saudi Arabia:**

AI initiatives in Saudi Arabia often need recurring compute capacity. DCP provides a local routing-first option with SAR-native billing so teams can keep spend planning and policy review in one flow.

**For GPU Owners (Providers):**

Register your NVIDIA GPU → install a lightweight daemon → jobs are matched to your node based on demand and availability.

Payout timing and withdrawal settings are shown in provider dashboard controls. No crypto.

**For Developers and ML Engineers (Renters):**

Submit LLM inference jobs via REST API or our Python/JS SDKs. Browse the GPU marketplace, pick your compute tier, and pay per job in SAR. The VS Code extension (Ctrl+Shift+V) connects your editor to a live vLLM inference session through DCP.

**PDPL Compliance:**
Operational routing and data handling are designed for Saudi-aligned residency and retention expectations, with controls documented in platform policy.

**Where we are:**
- Provider registration: live at dcp.sa/setup
- Renter registration: live at dcp.sa/renter/register
- Full API documentation: dcp.sa/docs

We're looking for early providers and developers who want transparent, Saudi-focused, dashboard-driven settlement.

If that's you — or if you know someone in KSA with idle GPU hardware — share this post and tag them.

🔗 dcp.sa

ثورة الذكاء الاصطناعي تحتاج حوسبة. اليوم، كثير من الفرق في المملكة تستخدم منصات عالمية، ثم تعيد ترتيب خيارات الفوترة وحوكمة البيانات محليًا. DCP يبني السعة محليًا، على أجهزة موجودة بالفعل في المملكة، بأسعار بالريال السعودي.

مواعيد وطرق السحب معروضة في إعدادات المزوّد. بدون عملات رقمية.

**للمطوّرين ومهندسي الذكاء الاصطناعي (المستأجرون):**

أرسل مهام الاستدلال عبر REST API أو حزم Python/JS. تصفّح سوق GPU، اختر مستوى الحوسبة المناسب، وادفع لكل مهمة بالريال. إضافة VS Code (Ctrl+Shift+V) تربط محررك مباشرة بخادم استدلال vLLM حي يعمل على DCP.

## Usage Notes

- **Format:** Long-form LinkedIn posts perform best with line breaks every 2–3 sentences — the formatting above is intentional
- **Audience targeting:** Post to Saudi tech groups (KACST alumni, Saudi Engineering Society, local AI meetup groups)
- **Best time:** Sunday or Monday morning (KSA), 9–11 AM AST
- **Visual:** Attach a graphic with the earnings table or a clean DCP logo + tagline card
- **Engagement:** In the first hour, respond to every comment — LinkedIn rewards early engagement with algorithmic reach
- **Tag strategy:** Avoid tagging competitors; do tag Saudi accelerators (SCAI, Monshaat) if there's a relationship
- **Cross-post:** Share the EN and AR versions as separate posts (not combined) for cleaner reach in each audience
